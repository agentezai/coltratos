## Iconography

The mocks consistently use **outline / stroked icons at 1.75 px stroke weight** with **round line caps and joins**. The character matches **Lucide** almost exactly.

We **do not** ship our own icon font, sprite, or `lucide-react` dependency (ADR-018). Instead, the registry at [src/components/ui/icon.tsx](../../src/components/ui/icon.tsx) holds 28 inline path definitions; `IconName = keyof typeof PATHS` is the typed name union.

To **add a new icon**:

1. Find or author the path data (Lucide's source is the upstream reference; preserve `viewBox="0 0 24 24"`, stroke-width 1.75, round caps + joins).
2. Append one entry to `PATHS` in `icon.tsx`.
3. Add the new name to the `Expected` union in `src/components/ui/__tests__/icon.test-d.ts`.
4. Run `npm run test` — the type-test confirms the union and registry stay in sync.

To **see all 28 registered names**, browse to **UI / Icon / Gallery** in this Storybook.

## Casing rule

The wordmark **COLTRATOS** is the only ALL-CAPS text in the system — a logotype, not running text. Everything else (page titles, button labels, table headers, eyebrows) is sentence case.

## Voice (Spanish, Colombia)

Second-person informal — **tú**, never *usted*. Friendly, direct, slightly warm without being cute. The product greets by first name (“¡Hola, María! 👋”) but never uses slang.

- ✅ *“Sabe en 1 minuto si puedes presentarte a un proceso del SECOP II.”*
- ✅ *“Aquí tienes el resumen de tu actividad.”*
- ✅ *“Tu archivo se analiza de forma privada y segura.”*
- ❌ *“El usuario puede cargar el pliego de condiciones.”* (too formal / third-person)

Two emoji are sanctioned, contextually:

- `👋` after a greeting
- `🇨🇴` next to the *“Plataforma colombiana”* chip on marketing

Never in body copy, tables, or buttons. Never decorative.

## Casing for the wordmark

The lockup SVG and the Sidebar's brand block both render `COLTRATOS` in uppercase. Headings in product copy stay sentence case; only the literal logotype is uppercase.
