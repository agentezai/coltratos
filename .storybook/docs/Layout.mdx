# Layout — spacing · radii · shadow

4-pt base scale. Cards at 12 px radius. Pills fully rounded. Shadows low and airy.

## Spacing

```css
--space-0  --space-1  --space-2  --space-3  --space-4  --space-5
--space-6  --space-8  --space-10 --space-12 --space-16 --space-20
```

Maps to: 0 / 4 / 8 / 12 / 16 / 20 / 24 / 32 / 40 / 48 / 64 / 80 px.

## Shadow elevation

Most screens stick to `--shadow-sm`. `--shadow-md` for popovers, `--shadow-lg` for modals, `--shadow-xl` reserved for the hero traffic-light card. Pixel-snap is enforced (no fractional px).

## App layout vars

```css
--app-sidebar-w   244px
--app-header-h    72px
--app-content-max 1280px
--app-gutter      32px
```

Sidebar sticks at 244 px. Topbar is 72 px tall. Content centers at 1280 px on ultra-wide displays. Gutters at 32 px between cards in the main column.

## Motion

```css
--ease-out    cubic-bezier(0.16, 1, 0.3, 1)   /* exaggerated ease-out for entrances */
--ease-in-out cubic-bezier(0.65, 0, 0.35, 1)  /* state changes */
--dur-fast    120ms   /* hover / focus */
--dur-base    180ms   /* button states, toasts */
--dur-slow    280ms   /* modals, big transitions */
```

Entrance: 6–8 px vertical rise + fade. No spring, no bounce. Processing screen is the one place with more expressive motion (rotating ring, shimmer, gentle pulse).
