# Colors

The COLTRATOS palette has three layers: **anchor** (navy + graphite, structural), **accent** (royal blue, the only CTA hue), and **semáforo** (functional green / amber / red — not decorative). Plus six tints used exclusively for icon wells.

Token names mirror [docs/design-system/source/project/colors_and_type.css](../../docs/design-system/source/project/colors_and_type.css) byte-for-byte. The `:root` block in [app/globals.css](../../app/globals.css) is the runtime source; the parity test at `src/__tests__/token-parity.test.ts` enforces 1:1 alignment (NFR-04).

Sidebar, hero, modal veil. Always paired with `--fg-inverse` (white) or `--fg-on-dark-2` for body copy on dark.

## Tints (icon wells only)

Pastel circular wells `~44 × 44 px` with a 50% radius. Icon inside is the same hue at `-500` or `-600`. Never put a stroked icon on a saturated background.

<div className="flex gap-4 my-6">
  {[
    ["--tint-blue", "#e7eeff"],
    ["--tint-green", "#e4f7ec"],
    ["--tint-amber", "#fff2d6"],
    ["--tint-red", "#fde6e6"],
    ["--tint-violet", "#ece9fe"],
    ["--tint-sky", "#e0f2fe"],
  ].map(([tok, hex]) => (
    <div key={tok} style={{ display: "flex", flexDirection: "column", alignItems: "center", gap: 4 }}>
      <div style={{ background: hex, width: 44, height: 44, borderRadius: 999 }} />
      <span style={{ fontFamily: "JetBrains Mono, monospace", fontSize: 10 }}>{tok}</span>
    </div>
  ))}
</div>

## What we avoid

- ❌ Bluish-purple gradient backgrounds
- ❌ Colored left-border accents on cards
- ❌ Decorative use of the semáforo trio (always pair with text + icon, RN-007)
- ❌ Saturated backgrounds behind stroked icons
