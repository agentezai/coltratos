# Typography

**Geist** for display + UI (self-hosted, full static family + variable). **JetBrains Mono** for IDs, file metadata, and currency in dense tables.

ADR-016 captures the choice over Inter, Plus Jakarta Sans, Satoshi, Manrope. Self-hosting eliminates the third-party CDN round-trip on first paint; the variable file is preloaded in [app/layout.tsx](../../app/layout.tsx).

## Families

```css
--font-display: "Geist", ui-sans-serif, system-ui, ...
--font-sans:    "Geist", ui-sans-serif, system-ui, ...
--font-mono:    "JetBrains Mono", ui-monospace, ...
```

Display and sans share the same family — Geist's weight axis (100–900) makes a distinct display face unnecessary in v1.

## Weights

100 / 200 / 300 / 400 / 500 / 600 / 700 / 800 / 900 — every weight has a self-hosted `.ttf` at `public/fonts/Geist-*.ttf` plus the variable file as progressive enhancement.

In practice we use:

- **400** body
- **500** emphasis (button labels, active nav)
- **600** section titles, card heads
- **700** H1 / numeric KPIs
- **800** landing display

## Casing

The wordmark **COLTRATOS** is the only ALL-CAPS — it is a logotype, not running text. Page titles, button labels, table headers, eyebrows are sentence case.
